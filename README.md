# bank-customer-value-prediction-System
An end-to-end Bank Customer Prediction project using Machine Learning. Includes data analysis (Pandas), model training (Scikit-learn), and a user-friendly Tkinter GUI for real-time predictions.

 🚀 Features
- 📊 Data preprocessing and cleaning
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning model using Scikit-learn
- 💾 Model saving using Joblib
- 🖥️ GUI-based prediction system using Tkinter

 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Tkinter (for GUI)

 📂 Project Structure
bank-customer-project/
│
├── data/ # Dataset
├── notebook/ # Jupyter Notebook (EDA + Training)
├── model/ # Saved ML model files
├── app/ # Application (main.py)
├── README.md
└── requirements.txt

⚙️ How to Run
 1. Clone the repository  git clone https://github.com/your-username/bank-customer-value-prediction-System.git
 2. Navigate to project  cd bank-customer-project/app
 3. Install dependencies
 4. Run the application

 📊 Model Details
- Algorithm: Random Forest Classifier
- Trained using Scikit-learn
- Handles customer demographic and financial data
